# American Dad! Apocalypse Soon - Usability Bug Report

## Title
Clone characters become difficult to interact with when event stand occupies front yard space.

## Environment
- Platform: iOS
- Game: American Dad! Apocalypse Soon
- Account: Nektarin

## Severity
Minor

## Priority
Medium

## Type
Usability Bug

## Description
When a large number of clones are present and a temporary event stand is placed in the front yard, available space becomes limited. As a result, clone characters are crowded together and some of them become difficult or impossible to tap.

## Screenshot

![Bug Screenshot](clone_bug.png)

## Steps to Reproduce
1. Launch the game.
2. Progress until a large number of clones are accumulated.
3. Wait for an in-game event that places a temporary stand in the front yard.
4. Observe clone placement and attempt to interact with all clones.

## Expected Result
All clones should remain accessible and easy to interact with regardless of active events.

## Actual Result
Some clones overlap with interface elements and become difficult or impossible to tap.

## Impact
Reduces usability and may prevent players from efficiently collecting rewards from clones.

## Report Status
Submitted to Reliance Games Support Team.
